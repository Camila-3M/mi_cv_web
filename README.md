# Mi CV Web

Este proyecto es una **Hoja de Vida (Currículum Vitae) interactiva** desarrollada con **HTML, CSS y JavaScript**, que muestra de forma organizada la información personal, formación académica, experiencia, proyectos y habilidades de una persona.

## Características
- Diseño limpio y responsivo utilizando **Flexbox**.  
- Interactividad básica con **JavaScript** (mostrar/ocultar secciones y efectos al pasar el mouse).  
- Navegación interna mediante un **menú con enlaces**.  
- Uso de **etiquetas semánticas y atributos accesibles** para una mejor comprensión y compatibilidad.  

## Propósito
Este proyecto sirve como base para crear un **CV web personalizable**, aplicando buenas prácticas de separación de responsabilidades y fundamentos del desarrollo front-end.
